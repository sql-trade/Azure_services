  .cpl - Commands 

 1.   control admintools                           The command is used to open the Administrative Tools (Windows Tools) folder in Control Panel  
 2.   control desktop                              The command helps to Control Personalization settings                                         
 3.   control schedtasks or  taskschd.msc          The command is used to open the task scheduler                                                
 4.   control userpasswords or control.exe /name   The command helps to manage the current Microsoft.UserAccounts/ user account                  
 5.   control userpasswords2                       The command is used to open the User Accounts dialog box                                      
 6.   dccw                                         The command helps to access the Control Display Color Calibration                             
 7.   desk.cpl                                     The command helps to access the Control Display settings                                      
 8.   devicepairingwizard                          The command helps to add a Device Wizard                                                      
 9.   mobsync                                      The command is used to open Microsoft Sync Center                                             
10.   recdisc                                      The command helps to Create a System Repair Disc                                              
11.   rstrui                                       The command helps to access the System Restore feature                                        
12.   sdclt                                        The command helps to access Devices and Printersâ€™ properties                                  
13.   shrpubw                                      The command helps to Create A Shared Folder Wizard                                            
14.   slui                                         The command helps to view and change Windows Activation settings                              
15.   systempropertiesdataexecutionprevention      The command is used to open Data Execution Prevention (DEP) feature                           
16.   systempropertiesperformance                  The command helps to access Performance Options                                               
17.   tabletpc.cpl                                 The command helps to access Pen and Touch options                                             
18.   UserAccourtControlSettings                   The command helps to adjust User Account Control (UAC) Settings                               
19.   wf.msc                                       The command helps to access Windows Firewall with Advanced Security                           
20.   wfs                                          The command is used to open Windows Fax and Scan utility
